키워드(카테고리) 관리
키워드 순서가 기본 우선순위를 결정합니다(취업→공부→게임…). 드래그로 순서 변경 가능.
목록을 드래그해서 순서를 바꾸면, 기본 우선순위 순서가 바뀝니다.
시간/레이아웃 설정
시간 미입력 마감은 ‘해당 날짜 + 컷오프’까지로 처리됩니다. (예: 컷오프 02:00이면 하루가 새벽 2시에 끝남)
화면 크기를 줄여도 달력 “크기감”을 유지하려고 셀 높이를 기본 고정합니다(높낮이 튐 방지).
클라우드(실시간 동기화) 설정
Firebase(구글 로그인 + Firestore)로 기기 간 실시간 동기화합니다. (최초 1회)
1) Firebase 콘솔에서 프로젝트 생성 → Authentication(구글) 활성화 → Firestore 생성
2) 프로젝트 설정의 firebaseConfig를 아래에 붙여넣기
3) 저장 후 새로고침(F5) → 로그인하면 PC/태블릿/모바일이 동일 데이터로 실시간 동기화됩니다.
• GitHub Pages(https)에서 열어야 로그인/동기화가 정상 동작합니다.
• Firebase Authentication의 Authorized domains에 kannad031.github.io 같은 도메인을 추가해야 합니다.
