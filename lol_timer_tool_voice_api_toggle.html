LoL 타이머 툴

즉시 실행 중심 경량 버전. 자동 네트워크 로딩 없이 열리고, 필요할 때만 설정 탭에서 업데이트 확인.

즉시 실행 모드
자동 로딩 없음

상대 픽 세팅

순서대로 TOP / JG / MID / ADC / SUP. 먼저 여기서 챔피언과 스펠을 잡아두면 음성 인식이 이 기준으로 해석.

스킬 정보

내 챔피언, 맞라인 상대, 궁 쿨 차이와 챔피언 스킬 쿨타임을 확인.

궁 비교 결과를 불러오는 중...

스킬 쿨타임 정보

정확한 챔피언명을 입력하면 현재 적용 버전 기준 Q / W / E / R 쿨타임을 표시.

빠른 시작 모드
설정 탭에서 업데이트 확인 후 API 기능을 켜면 사용할 수 있음.

게임

숫자 입력 + 음성 입력 둘 다 가능. 예: 849 → 08:49, 음성은 “트린 플 17분 즈음 썼다”처럼 압축해서 말해도 해석 시도.

현재 게임 시간
00:00

음성 입력

대기 중
크롬/엣지에서 1회 듣기 방식으로 동작.
아직 인식 결과 없음

궁 스킬가속 계산기

아이템/룬과 수기 입력으로 계산한 값을 선택한 포지션에 반영.
총 궁 스킬가속: 0

음성 예시

- 트린 플 1701
- 탑 트린다미어 점멸 17분 01초 사용
- 미드 아리 궁 14분 20초
- 서폿 점화 9분대 사용
